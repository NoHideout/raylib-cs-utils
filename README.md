# raylib-cs-utils

A collection of reusable C# utilities and modular systems for Raylib-cs and rlImGui-cs. This project is built on top of my [Raylib/ImGui template](https://github.com/NoHideout/raylib-cs-imgui-template) and acts as a sandbox to test the utilities.

*Note: This repository is currently under active development. I will add Features step-by-step.*

## Usage
For now add this repo as a submodule to your project.
The Usage of individual modules will be added.

## Dependencies
* [Raylib-cs](https://github.com/raylib-cs/raylib-cs) - C# bindings for [Raylib](https://github.com/raysan5/raylib)
* [rlImgui-cs](https://github.com/raylib-extras/rlImGui-cs) - C# integration for [Dear ImGui](https://github.com/ocornut/imgui)
